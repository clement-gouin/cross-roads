# Cross-Roads
*Where will you go next?*

### [Tool link](https://clement-gouin.github.io/cross-roads/)

## Data format

Format is made line by line

Header (1 line):
```txt
1   Title (html)
```

After that each link is defined as follows:
```txt
1   link
2   Label (html)
```
